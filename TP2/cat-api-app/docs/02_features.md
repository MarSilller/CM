# Application Features
1. **Feline Gallery:** Fetch and display cat images in a grid using a RecyclerView.
2. **Breed Details:** View detailed info (name, origin, temperament, description) for each cat.
3. **Swipe-to-Refresh:** Fetch a new batch of random cat images.
4. **FIFO Favorites:** A "Top 5" favorite system where the oldest favorite is removed when a 6th is added.
5. **Smart Cache:** Store up to 50 items locally. Maintain a buffer of 10 items ahead/behind the current scroll position.
6. **Offline Mode:** Seamlessly view cached cats and favorites when no internet is available.
